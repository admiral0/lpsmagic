\subsection{\-Detailed \-Description}
\-Manages all renderers. 

\-All renderers should be registered so the config tokens can be parsed and appropriate plugin can be called.\par
 \-The tokens are in the form of\-:\par
 \par
\par
{\ttfamily \%}\mbox{[}render\-Options\mbox{]}prefix\{arg1\}\{arg2\}\par
\par
 \-Render options can be these separated by \char`\"{}$|$\char`\"{}\-:
\begin{DoxyItemize}
\item position=x,y -\/$>$ \-Absolute positioning 
\item margin=top,right,bottom,left -\/$>$ \-Margins 
\begin{DoxyItemize}
\end{DoxyItemize}\par
\par
 \-Arguments can be arrays (comma separated vars). \-There cannot be 0 arguments, but prefix\{\} is acceptable \begin{DoxyNote}{\-Note}
\-This is a singleton class 
\end{DoxyNote}

\end{DoxyItemize}
