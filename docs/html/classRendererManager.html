Detailed Description

Manages all renderers.

All renderers should be registered so the config tokens can be parsed and appropriate plugin can be called.
The tokens are in the form of:


%[renderOptions]prefix{arg1}{arg2}

Render options can be these separated by "|":

  • position=x,y -> Absolute positioning
  • margin=top,right,bottom,left -> Margins


    Arguments can be arrays (comma separated vars). There cannot be 0 arguments, but prefix{} is acceptable
    Note:
    This is a singleton class

QImage RendererManager::render ( )

Render the image to be displayed.

The various lines are parsed and printed onto the image all tokens are composited into the image.
There are 2 "fake" renderers:

  • %newline{}
  • %spacer{pixels}
Note:
This function can be used to generate previews.
